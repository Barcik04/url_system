# Limitations

## Can a guest user access the dashboard?
No. Only signed-in users can access the dashboard.

## Can a guest user create a short URL?
Yes. A guest user can create a short URL without signing in.

## What is the limitation of creating a short URL as a guest?
If a guest user creates a short URL, it will not be available later in a personal dashboard. The user should copy and save it immediately.

## Can a signed-in user manage URLs created by other normal users?
No. A signed-in user can manage only their own URLs.

## Can every signed-in user access the admin panel?
No. Only users with admin access can open the admin panel.

## What is the link limit for the Regular plan?
The Regular plan allows up to 10 links.

## What is the link limit for the Premium plan?
The Premium plan allows unlimited links.

## What is the minimum password length?
The password must be at least 8 characters long.

## Can a user sign in without verifying the email address?
No. The account must be verified through email before sign-in can work.

## What happens if the user sends too many requests in a short time?
The application may temporarily block further requests. The user should wait a couple of minutes and try again later.

## Can a deleted account be recovered?
No. Account deletion is irreversible.

## Can the user avoid copying a guest-created short URL?
No. A guest user should copy and save the generated short URL immediately, because it will not be stored in a personal dashboard.

## Are all application features available to guests?
No. Some features are available only to signed-in users, and some sections are available only to admins.
