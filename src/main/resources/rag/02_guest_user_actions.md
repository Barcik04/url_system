# Guest User Actions

## How does a guest user register?
1. Click **Sign up** in the top navigation bar.
2. Enter an email address.
3. Create a password with at least 8 characters.
4. Submit the registration form.
5. Open the verification email sent to the provided address.
6. Click the verification button in the email.
7. After email verification, the account is ready for sign-in.

## How does a guest user sign in?
1. Click **Sign in** in the top navigation bar.
2. Enter the same account details used during registration.
3. Submit the sign-in form.
4. After successful sign-in, the user can use features available to signed-in users.

## How does a guest user create a short URL?
1. Click the application logo in the top-left corner of the screen.
2. A URL creation form will appear.
3. Paste the original URL into the input field.
4. Optionally set an expiry date.
5. Click **Create URL**.
6. A new short URL will appear at the bottom of the screen.
7. Copy and save the short URL immediately.

## Important note for guest users creating short URLs
If a user creates a short URL without being signed in, that URL will not be available later in a personal dashboard. The user should copy and save it immediately.
