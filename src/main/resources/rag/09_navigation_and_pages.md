# Navigation and Pages

## What can a guest user access from the top navigation bar?
A guest user can use the top navigation bar to:
- open **Sign up**
- open **Sign in**
- click the application logo to open the short URL creation form

## What happens when the user clicks the application logo?
Clicking the application logo opens the short URL creation form on the main screen.

## Where can a signed-in user find the main menu?
A signed-in user can find the main menu by clicking the three-line menu icon in the top-right corner of the navigation bar.

## What pages can a signed-in user open from the main menu?
A signed-in user can open:
- **Dashboard**
- **Settings**

Users with admin access can also open:
- **Admin**

## What is the Dashboard page used for?
The Dashboard page is used for viewing and managing URLs created by the signed-in user.

## What can the user do on the Dashboard page?
On the Dashboard page, the user can:
- view their own URLs
- edit their own URLs
- delete their own URLs

## What is the Settings page used for?
The Settings page is used for managing profile and account options.

## What can the user do on the Settings page?
On the Settings page, the user can:
- change their avatar
- open the subscriptions section
- delete their account

## What is the Subscriptions section used for?
The Subscriptions section is used for viewing available subscription plans, buying Premium, and canceling an active Premium subscription.

## What is the Admin page used for?
The Admin page is used for managing all URLs in the system.

## Who can access the Admin page?
Only users with admin access can open the Admin page.

## What can an admin do on the Admin page?
An admin can:
- view all URLs in the system
- create URLs
- edit URLs
- delete URLs
- filter URLs
