# General Rules

## Who can access the dashboard?
Only signed-in users can access the dashboard.

## How many links can a user with the Regular plan have?
A user with the Regular plan can have up to 10 links.

## How many links can a user with the Premium plan have?
A user with the Premium plan can have unlimited links.

## What are the password requirements?
Passwords must be at least 8 characters long.

## What should the user do after too many requests?
If the user sends too many requests and gets blocked, they should wait a couple of minutes and try again later.
