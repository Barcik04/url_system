# Common Problems

## Why can’t the user sign in after registering?
The user must verify the account through the email verification message before sign-in can work.

## What should the user do if they did not verify the email yet?
The user should open the verification email and click the verification button first.

## Why can’t the user create more links?
The user may have reached the link limit of their current plan. Regular users can have up to 10 links, while Premium users can have unlimited links.

## What should the user do if they are rate-limited?
If too many requests were sent in a short time, the user should wait a couple of minutes and try again later.

## Why should a guest user save the generated short URL immediately?
Because guest-created URLs are not stored in a personal dashboard for later access.
