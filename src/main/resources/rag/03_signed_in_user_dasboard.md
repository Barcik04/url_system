# Signed-In User Dashboard

## How does a signed-in user access the dashboard?
1. Click the three-line menu icon in the top-right corner of the navigation bar.
2. Select **Dashboard** from the menu.
3. The dashboard will display URLs created by the signed-in user.

## What can a signed-in user do in the dashboard?
A signed-in user can:
- view their own URLs
- edit their own URLs
- delete their own URLs

## How does a signed-in user edit a URL?
1. Open the **Dashboard**.
2. Move the mouse over the URL that should be edited.
3. On the right side of that URL row, click the **pencil** icon.
4. The edit form will open.
5. Update the URL field if needed.
6. Update the expiry date if needed.
7. Save the changes.

## How does a signed-in user delete a URL?
1. Open the **Dashboard**.
2. Move the mouse over the URL that should be deleted.
3. On the right side of that URL row, click the **trash can** icon.
4. Confirm the deletion.

## What happens if the user leaves fields empty in the URL edit form?
If the user does not want to update a field, they can leave it empty.
