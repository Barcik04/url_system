# Admin Panel

## How does an admin access the admin panel?
1. Click the three-line menu icon in the top-right corner of the navigation bar.
2. Select **Admin** from the menu.

## What does the admin panel show?
The admin panel shows all URLs existing in the system.

## What actions are available in the admin panel?
In the admin panel, an admin can:
- create URLs
- edit URLs
- delete URLs
- filter URLs

## Are admin panel actions similar to dashboard actions?
Yes. Admin panel URL operations work similarly to the normal user dashboard, but the admin panel contains all system URLs.
